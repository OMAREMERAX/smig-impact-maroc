# 📊 SMIG Impact on Unemployment in Morocco

## 🚀 Project Overview
This project analyzes the causal impact of minimum wage (SMIG) increases on unemployment in Morocco using econometric methods.

## 🧠 Methodology
- Difference-in-Differences (DiD)
- Regression analysis (Statsmodels)

## 📊 Data
- SMIG (2015–2023)
- Unemployment rates
- Sectors: services vs industry

## 📈 Results
The results show no significant causal effect of SMIG on unemployment.
